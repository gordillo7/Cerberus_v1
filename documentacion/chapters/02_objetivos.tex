El principal objetivo de este Trabajo de Fin de Grado consiste en desarrollar una herramienta que automatice las tareas asociadas a la fase de reconocimiento de una auditoría, facilitando y agilizando la recopilación de información sobre un objetivo. Además, esta herramienta deberá ser capaz de generar informes automáticamente a partir de los resultados obtenidos, así como incorporar un asistente de inteligencia artificial para proporcionar recomendaciones contextualizadas que asistan al usuario durante el proceso de auditoría.

\section{Subobjetivos específicos}
Como consecuencia de este objetivo principal, se identifican los siguientes subobjetivos específicos, organizados según las tres funcionalidades clave de la herramienta:

\subsection*{Desarrollo de la herramienta}
\begin{itemize}
  \item Estudiar la metodología seguida en la fase de reconocimiento de una auditoría de seguridad.
  
  \item Analizar las distintas alternativas tecnológicas disponibles para cada tarea de reconocimiento, con el fin de seleccionar las opciones más adecuadas en términos de rendimiento, flexibilidad e integración.
  
  \item Desarrollar módulos individuales responsables de cada una de las tareas clave que componen la fase de reconocimiento.
  
  \item Orquestar las herramientas seleccionadas bajo una única solución que gestione y controle el flujo de ejecución entre los diferentes módulos.
  
  \item Permitir la ejecución de la herramienta mediante consola, ampliando su adaptabilidad a distintos perfiles de usuario.
  
  \item Diseñar una arquitectura modular y escalable que permita futuras ampliaciones y facilite el mantenimiento del sistema.
  
  \item Implementar una interfaz gráfica amigable que simplifique el uso de la herramienta, incluso para usuarios con conocimientos técnicos limitados.
  
  \item Incorporar un sistema de gestión de objetivos basado en proyectos, con el objetivo de mejorar la organización y el seguimiento de los análisis realizados.
  
  \item Ofrecer distintas opciones de configuración para que la herramienta pueda ajustarse a distintos escenarios de análisis y niveles de profundidad.
\end{itemize}

\subsection*{Generación automática de informes}
\begin{itemize}
  \item Desarrollar un sistema capaz de generar informes técnicos de forma automática a partir de los resultados obtenidos, presentando la información de manera clara y visual.

  \item Incluir un sistema de versiones que permita mantener un histórico de todos los informes generados por objetivo.
\end{itemize}

\subsection*{Integración de inteligencia artificial}
\begin{itemize}
  \item Explorar las características y capacidades de diferentes modelos de inteligencia artificial con el objetivo de identificar aquel que mejor se adapte a las necesidades de este proyecto.

  \item Incorporar un asistente de inteligencia artificial que proporcione asistencia contextual al usuario y sugerencias sobre acciones posteriores a los resultados obtenidos.
\end{itemize}
