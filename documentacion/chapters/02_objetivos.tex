El principal objetivo de este Trabajo de Fin de Grado consiste en desarrollar una herramienta que \textbf{automatice} las tareas asociadas a la fase de reconocimiento de una auditoría, facilitando y agilizando la \textbf{recopilación de información} sobre un objetivo. Con ello se pretende \textbf{reducir significativamente el tiempo}, la carga de trabajo y los recursos humanos necesarios para realizar dichas tareas de forma manual, al mismo tiempo que se proporciona una \textbf{interfaz gráfica} intuitiva, fácil de usar y con distintas opciones de configuración que la hagan accesible también a perfiles sin conocimientos técnicos avanzados.

\section{Subobjetivos específicos}
Como consecuencia de este objetivo principal, se identifican los siguientes \textbf{subobjetivos específicos}:

\begin{itemize}
  \item Estudiar la \textbf{metodología} seguida en la fase de reconocimiento de una auditoría de seguridad.
  
  \item Analizar las distintas \textbf{alternativas tecnológicas} disponibles para cada tarea de reconocimiento, con el fin de seleccionar las opciones más adecuadas en términos de rendimiento, flexibilidad e integración.
  
  \item Desarrollar \textbf{módulos individuales} responsables de cada una de las tareas clave que componen la fase de reconocimiento.
  
  \item \textbf{Orquestar} las herramientas seleccionadas bajo una única solución que gestione y controle el flujo de ejecución entre los diferentes módulos.
  
  \item Diseñar una arquitectura \textbf{modular y escalable} que permita futuras ampliaciones y facilite el mantenimiento del sistema.
  
  \item Implementar una interfaz gráfica amigable que simplifique el uso de la herramienta, incluso para usuarios con \textbf{conocimientos técnicos limitados}.
  
  \item Incorporar un sistema de gestión de objetivos basado en \textbf{proyectos}, con el objetivo de mejorar la organización y el seguimiento de los análisis realizados.
  
  \item Integrar un sistema de \textbf{inteligencia artificial} que proporcione asistencia contextual al usuario y sugerencias sobre acciones posteriores a los resultados obtenidos.
  
  \item \textbf{Generar informes} técnicos de forma automática a partir de los resultados del reconocimiento, presentando la información de manera clara y visual.
  
  \item Permitir la ejecución de la herramienta mediante \textbf{consola}, ampliando su adaptabilidad a distintos perfiles de usuario.
  
  \item Ofrecer distintas \textbf{opciones de configuración} para que la herramienta pueda ajustarse a distintos escenarios de análisis y niveles de profundidad.
\end{itemize}
