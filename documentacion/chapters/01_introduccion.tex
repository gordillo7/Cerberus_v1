Esta plantilla sirve como ejemplo de TFE. Las secciones son las que están en la normativa, aunque si el tutor lo considera se puede cambiar la estructura. Esta sección la aprovechamos para introducir brevemente \LaTeX.
\par

\section{Secciones}
Las secciones se crean con \textit{section}. Las subsecciones y subsubsecciones con \textit{subsection} y \textit{subsubsection}, respectivamente. Si se desea que alguna subsección en concreto no salga en el índice se pueden usar los mismos comandos añadiéndoles un asterisco al final (\textit{subsection*} y \textit{subsubsection*}).
\par
\lipsum[1][1-4]
\subsection{una subsección}
\lipsum[1][1]
\subsection{otra subsección}
\lipsum[1][2]
\subsubsection{subsubsecciones}
¡Por defecto las subsubsecciones no aparecen en el índice! Este comportamiento se puede cambiar, pero lo recomendable es organizar los capítulos en 3 niveles para facilitar la lectura.

\section{Citas y referencias}
Para citar un texto hay que incluirlo en el fichero \textit{LocalBibliography.bib} de esta plantilla. Una vez hecho se puede referenciar usando el comando \textit{cite}~\cite{LaTeX_tutorials}.
Para referenciar imágenes, secciones o tablas se usa el comando~\textit{autoref}. Por ejemplo~\autoref{fig:logoEpcc} o~\autoref{sec:objetivos}.
\par
Para facilitar la lectura, cuando se insertan citas y referencias es conveniente insertar un carácter de espacio sin salto de línea \textit{\~} antes del comando de cita o referencia.

\section{Código}
El código se puede insertar así: 
% linenos indica si se muestra el número de línea
\begin{minted}[linenos=false]{bash}
echo "Este es un ejemplo de cómo se inserta código con la plantilla de Trabajo Fin de Estudios de la Escuela Politécnica de Cáceres"
\end{minted}

También se puede seleccionar desde un fichero, por ejemplo:
%\inputminted[firstline=1,lastline=7]{cpp}{code/example.cpp}

Los lenguajes reconocidos para marcar se pueden consultar en \url{https://pygments.org/docs/lexers/}. Aunque depende del tipo de trabajo, se recomienda incluir código sólo en anexos.
