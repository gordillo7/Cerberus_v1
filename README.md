# **PenTest Helper**

Una herramienta diseñada para automatizar las tareas básicas de pentesting, como el escaneo de puertos, ataques de fuerza bruta y generación de reportes.

## **Características**

- **Escaneo inicial con Nmap**: Detecta puertos abiertos y servicios activos.
- **Ataques de Fuerza Bruta**:
  - **FTP**: Acceso anónimo y fuerza bruta con diccionarios.
  - **WordPress**: Enumeración de usuarios y fuerza bruta de contraseñas.
- **Orquestador Central**: Gestiona la secuencia de ejecución de los módulos según los servicios detectados.
- **Reportes Automatizados**: Consolida hallazgos en un informe final.
